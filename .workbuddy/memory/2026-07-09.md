# 2026-07-09 工作日志

## 微信记账小程序 - UI 优化与功能增强

### 用户反馈与修复
1. **默认头像**
   - 根据用户提供的机器人厨师猫参考图，复制为 `miniprogram/images/default-avatar.png`
   - `login` 云函数新用户默认昵称改为「记账小助手」，头像指向该图片

2. **记账页金额显示不完整**
   - 修复 `add.wxss` 中 `.amount-input` 缺少高度/行高导致 placeholder 文字被截断
   - 增加 `height: 120rpx; line-height: 120rpx; padding: 0;`

3. **记账页首次进入无分类**
   - 新增 `initCategories(type)` 方法，在 `onLoad` 时主动初始化支出分类
   - `switchType` 也复用该方法，确保分类数据正确写入

4. **邀请好友流程优化**
   - 账本详情页点击「邀请」弹出美观邀请弹窗（账本图标、名称、邀请码、成员头像、复制按钮、分享按钮）
   - 分享按钮使用 `open-type="share"`，触发 `onShareAppMessage` 返回邀请卡片
   - 优化分享文案：title + desc + 邀请码路径
   - `ledger.js` 支持从分享卡片 path 读取 `code` 参数，自动提示加入账本

5. **订阅消息推送优化**
   - 首页 `onShow` 中增加订阅授权引导：未开启提醒时每日首次弹出提示
   - 用户确认后调用 `wx.requestSubscribeMessage` 并自动开启 remindEnabled
   - `remindCheck` 增加近 24 小时成就解锁查询日志，为后续多模板推送做准备
   - 已告知用户：个人主体无法强制推送，订阅消息为一次性授权，需企业主体才能长期订阅/强制推送
