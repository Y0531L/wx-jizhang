# 微信记账小程序

一个功能完整的微信记账小程序，支持微信支付自动记账、多平台账单识别、好友排行、成就系统、多人协同记账。

## 功能特性

### 1. 微信支付自动记账
- 小程序内微信支付成功后，通过支付回调云函数自动创建账单
- 支持绑定支付订单与账单分类，自动归类

### 2. 多平台账单识别提醒
- 支持上传支付宝 / 云闪付 / 银行 App 的账单截图
- 通过 OCR + 正则提取自动识别金额、时间、商户
- 识别结果生成「待补记」提醒，用户确认后一键记账
- 定时订阅消息提醒（每天早 8 点、晚 8 点）

> ⚠️ 微信小程序运行在沙箱中，无法直接监听系统通知。本方案通过截图识别 + 定时提醒实现等效功能。

### 3. 好友排行与成就系统
- 好友排行榜：支持本周 / 本月 / 累计三个周期
- 四个维度：谁最能花、谁最能赚、谁最能攒、谁最勤快
- 成就系统：18+ 成就徽章，覆盖记账、坚持、理财、社交、特殊时刻
- 等级体系：通过记账积累经验，升级称号

### 4. 多用户协同记账
- 创建共享账本（家庭 / 室友 / 团队 / 旅行）
- 邀请码加入或分享小程序卡片邀请
- 账本内成员共享账单，实时同步
- 成员管理：创建者可移除成员、重命名、解散账本

### 5. 数据统计
- 近 6 个月收支趋势（柱状图）
- 分类占比（环形图）
- 近 3 个月收支对比
- 按月查看账单，按日分组

### 6. 其他
- 账单导出 CSV
- 自定义分类管理
- 个人 / 共享账本自由切换

## 技术栈

| 层 | 技术 |
|----|------|
| 前端 | 微信小程序原生（WXML + WXSS + JS） |
| 后端 | 微信云开发（CloudBase） |
| 数据库 | 云开发文档数据库（NoSQL） |
| 存储 | 云开发云存储（截图） |
| 消息 | 微信订阅消息（记账提醒） |
| 支付 | 微信支付（自动记账） |

## 快速开始

### 1. 克隆项目

```bash
git clone <your-repo-url>
cd wx记账小程序
```

### 2. 导入微信开发者工具

- 打开微信开发者工具
- 选择「导入项目」
- 项目目录选择本文件夹
- AppID 填入你的小程序 AppID（或使用测试号）

### 3. 配置云开发

- 开通云开发，获取环境ID
- 替换 `miniprogram/app.js` 中 `globalData.cloudEnv` 的值
- 替换 `project.config.json` 中的 `appid`

### 4. 初始化数据库

详见 [docs/DATABASE.md](./docs/DATABASE.md)

### 5. 部署云函数

右键 `cloudfunctions` 下每个云函数，选择「上传并部署：云端安装依赖」

### 6. 体验

在开发者工具中预览，或扫码真机体验。

详细部署步骤见 [docs/DEPLOY.md](./docs/DEPLOY.md)
